# Motor Insurance Claims Dashboard

Interactive dashboard analysing 500 motor insurance claims across Kenya (2022-2023).

## Tools
- Python, Pandas, Plotly

## Files
- `dashboard.py` — full source code
- `motor_claims_dashboard.html` — interactive dashboard

## How to Run
1. Install dependencies: `pip install pandas plotly`
2. Run: `python dashboard.py`
3. Dashboard opens automatically in your browser
